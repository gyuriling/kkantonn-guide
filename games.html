🎮 /가챠

×

📖 사용법: /가챠 [확률] [재화] 선택사항:[단위] [최소] [최대]

입력한 확률로 가챠 시뮬레이션을 실행합니다.
매우 낮은 확률의 경우 실행 시간이 길어질 수 있으며, 깡통이가 자동으로 중단시키는 제한도 포함되어 있습니다.

  • ✔️ 단위 옵션으로 재화 이름 지정 가능 (예: “루나”)
  • ✔️ 최소·최대 개수 옵션으로 1회당 획득량 설정
  • ✔️ 매우 낮은 확률에 대한 시도 제한 처리

예시:
/가챠 0.01 100 단위:루나 → “총 237번 시도하여, 23,700루나 사용해써요!”

🎞️ /가챠 사용 예시 GIF 자리

🎮 /숫자야구

×

📖 사용법: /숫자야구 선택사항:[길이] [숫자]

랜덤으로 생성된 3자리 또는 4자리 숫자를 맞추는 게임입니다.
깡통이가 스트라이크(S), 볼(B)로 힌트를 제공합니다.

  • ✔️ 채널별로 정답 공유 → 여러 유저가 참여 가능
  • ✔️ 제한시간 120초, 시도 제한 10회
  • ✔️ 중복 없는 숫자만 허용

예시:
/숫자야구 길이:4 → “정답: 3951”

🎞️ /숫자야구 사용 예시 GIF 자리

🎮 /타자

×

📖 사용법: /타자 [언어]

랜덤 문장을 출력해 타자 속도를 측정합니다.
한국어, 영어, 일본어 중 선택 가능하며, 5초 카운트다운 후 문장을 이미지로 표시합니다.

  • ✔️ 오타 없이 입력해야 인정
  • ✔️ 초당 입력 속도 및 소요시간 출력
  • ✔️ 타자 종료 버튼으로 강제 종료 지원
  • ✔️ 여러 서버 동시 실행 가능

예시:
/타자 영어 → “속도: 4.2자/초, 소요시간: 13.5초”

🎞️ /타자 사용 예시 GIF 자리

🎮 /냥

×

📖 사용법: /냥 [숫자]

입력한 개수만큼 “냥” 문자열을 랜덤 출력합니다.
30% 확률로 “귤”도 섞여 출력됩니다.

예시:
/냥 12 → “냥냥귤냥냥냐아앙냥귤냥냥”

🎞️ /냥 사용 예시 GIF 자리

🎮 /퀴즈

×

📖 사용법: /퀴즈 [문제] [정답]

깡통이가 퀴즈를 출제하고, 다른 유저들이 참여할 수 있도록 합니다.
참여자는 “정답 제출” 버튼으로 답을 입력하고, 출제자는 언제든지 퀴즈를 종료할 수 있습니다.

  • ✔️ **정답 제출** 버튼: 정답 입력 창 표시
  • ✔️ **정답 확인** 버튼: 현재까지 맞힌 유저 목록 멘션 출력
  • ✔️ **제출 로그** 버튼: 모든 답변 시도 기록 표시
  • ✔️ **퀴즈 종료** 버튼: 출제자가 퀴즈를 강제 종료 가능
  • ✔️ 중복 정답 방지 → 한 명당 1회 인정
  • ✔️ 서버별로 1개의 퀴즈만 실행 가능

예시:
/퀴즈 수도는? 서울 → “정답자: @user1 @user2”

🎞️ /퀴즈 사용 예시 GIF 자리
