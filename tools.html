🧮 /계산

×

📖 사용법:
/계산 [수식]

🛠 옵션 설명

  • 수식 (필수): 계산할 수식 입력 (예: (5+3)^2)

입력한 수식을 계산합니다.
math.js 모듈로 작동하여 괄호와 다양한 함수가 포함된 복잡한 연산도 지원합니다.
기본 수학 함수 대부분 사용 가능 (예: sqrt(), sin(), log(), abs(), random())

💡 예시

/계산 (5+3)^2

→ 64

/계산 sqrt(16)

→ 4

🎲 /골라

×

📖 사용법:
/골라 [항목] 선택:[개수]

🛠 옵션 설명

  • 항목 (필수): 공백으로 구분된 단어 목록 (예: 치킨 피자 떡볶이)
  • 개수 (선택): 랜덤으로 선택할 단어 개수 (기본값: 1)

입력한 단어들 중에서 지정한 개수만큼 랜덤으로 선택합니다.
같은 단어가 여러 번 나올 수는 없습니다.

💡 예시

/골라 치킨 피자 떡볶이

→ 치킨

/골라 치킨 피자 떡볶이 개수:2

→ 피자, 떡볶이

📝 /글자수세기

×

📖 사용법:
/글자수세기 [텍스트]

🛠 옵션 설명

  • 텍스트 (필수): 글자 수를 셀 문자열 (예: 안녕하세요)

입력한 문자열의 공백 포함/제외 글자 수와 바이트 수를 계산합니다.
공백을 제외한 글자 수도 함께 표시됩니다.

/글자수세기 안녕하세요

공백포함 : 5 (15 byte)
공백제외 : 5 (15 byte) 

/글자수세기 hello world

공백포함 : 11 (11 byte)
공백제외 : 10 (10 byte)

🎲 /랜덤숫자

×

📖 사용법:
/랜덤숫자 [최소] [최대] [개수] 선택:[중복허용]

🛠 옵션 설명

  • 최소 (필수): 랜덤 숫자 범위의 시작값
  • 최대 (필수): 랜덤 숫자 범위의 끝값
  • 개수 (필수): 생성할 숫자의 개수
  • 중복허용 (선택): false 입력 시 중복 비허용 (기본값: 허용)

지정한 범위에서 랜덤 숫자를 생성합니다.
중복허용:false 옵션 사용 시 같은 숫자가 중복되지 않습니다.

💡 예시

/랜덤숫자 1 10 3

→ 랜덤 숫자(중복 허용): 3, 7, 1

/랜덤숫자 1 5 3 중복허용:false

→ 랜덤 숫자(중복 없음): 5, 2, 1

🎞️ /랜덤숫자 사용 예시 GIF 자리

🔤 /변환

×

📖 사용법:

🛠 옵션 설명

  • 입력 (필수): QWERTY 자판으로 입력한 문자열 (예: dkssud)

💡 예시

→ 안녕

🖼️ /프사

×

🛠 옵션 설명

🌎 /번역

×

📖 사용법:
/번역 [텍스트] 선택:[언어]

🛠 옵션 설명

  • 텍스트 (필수): 번역할 문장 입력
  • 언어 (선택): 번역할 대상 언어
    한국어(KO), 영어(EN), 일본어(JA), 중국어(간체)(ZH)
    지정하지 않으면 기본값: 한국어(KO)

입력한 문장을 지정한 언어로 번역합니다.
결과는 텍스트 형태로 반환됩니다.

💡 예시

/번역 텍스트:Hello 언어:KO

→ 안녕하세요

/번역 텍스트:ありがとう 언어:EN

→ Thank you

💱 /환율

×

📖 사용법:
/환율 [금액] [입력통화] [출력통화]

🛠 옵션 설명

  • 금액 (필수): 변환할 금액 입력
  • 입력통화 (필수): 현재 금액의 통화 단위
    지원: USD 🇺🇸, KRW 🇰🇷, EUR 🇪🇺, JPY 🇯🇵, CNY 🇨🇳, GBP 🇬🇧, AUD 🇦🇺, CAD 🇨🇦, CHF 🇨🇭, SGD 🇸🇬, THB 🇹🇭, MYR 🇲🇾
  • 출력통화 (필수): 변환할 대상 통화 단위
    지원: 위와 동일

입력한 금액을 지정한 환율로 변환합니다.
결과는 임베드 형태로 표시됩니다.

💡 예시 (임베드 본문)

🤍 환율 계산 🤍
1,000           USD 🇺🇸
        ⇌
1,312,500       KRW 🇰🇷
Footer: 1 USD = 1,312.500000 KRW
